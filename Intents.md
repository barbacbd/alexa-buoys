# Buoy 

The intent will provide users with information about a specific buoy.

## Utterances

- buoy report for {buoy_id}
- what is the buoy report for {buoy_id}
- {buoy_id} report
- {buoy_id} buoy report
- can you provide the buoy report for {buoy_id}

## Input

- buoy_id - ID or name of the buoy
